# ✅ Task Manager

Простой веб-менеджер задач на чистом **HTML / CSS / JavaScript** без сторонних фреймворков. Позволяет добавлять, редактировать, фильтровать и выполнять задачи в реальном времени.

## 🚀 Технологии

- **HTML5** — разметка страницы
- **CSS3** — стилизация и адаптивность
- **Vanilla JavaScript** — вся логика приложения

## 🗂 Структура проекта

```
Web/
├── mainPage.html   # HTML-разметка страницы
├── script.js       # Вся логика: создание, редактирование, фильтрация
└── style.css       # Стили интерфейса
```

## ✨ Функциональность

- **Добавление задачи** — указание названия, приоритета, типа и даты выполнения
- **Редактирование** — изменение названия и даты существующей задачи
- **Выполнение** — помечает задачу как завершённую (выделяется зелёным цветом)
- **Удаление** — удаление отдельных задач или всего списка
- **Фильтрация** — по приоритету (High / Medium / Low) и типу задачи
- **Сортировка** — по важности через чекбокс
- **Валидация** — проверка на пустые поля и корректность даты (только будущая)
- **Кнопка «Up»** — быстрое возвращение наверх страницы

## 📦 Запуск

Проект работает напрямую в браузере — никакой сборки не требуется.

1. Клонируй репозиторий:
```bash
git clone https://github.com/danechka0075/Web.git
cd Web
```
2. Открой `mainPage.html` в любом браузере.
